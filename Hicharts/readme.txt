Le répéteur

                               Le but de cet exercice est de coder un répéteur d’élément sous forme de directive (Attention, le but n’est pas de recoder, ni utiliser la directive ngRepeat d’Angular). Voici le code initial coté HTML

<div repeat=’5’>  ONCH  </div>

                               BONUS : Si tu es un chevalier du data-binding, tu peux rajouter un « input text » pour que l’utilisateur puisse rentrer le nombre de répétition de son choix.
